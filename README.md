This GitHub repository contains the code used to generate the biome-specific Vegetation Drought Index (VDI) dataset for India at 5 km spatial resolution for the period 2001–2023. The repository includes scripts for processing the input variables, constructing the VDI, and generating both the pan-India all-biome product and biome-specific outputs for forests, croplands, and grasses, including shrublands and savannas. The code is intended to support transparency, reproducibility, and reuse of the methods described in the associated Scientific Data manuscript. It also provides the workflow used to prepare the NetCDF outputs and input-variable layers for the entire Indian region. This repository can be used by researchers and practitioners interested in vegetation drought monitoring, biome-level drought assessment, ecological applications, and climate-impact studies. The code will be maintained as the public companion repository for the published dataset and manuscript.
